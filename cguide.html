Introduction

C is a multi-purpose procedural programming language that maps efficiently to general machine instructions and assembly language. It is widely used in computer operating systems and application software for computers. It was developed at Bell Labs in the 1970s and has since become one of the most widely used programming languages worldwide. C compilers are available for the majority of computer architectures and operating systems.

This guide will cover the basics of C syntax and functionality. Some basic familiarity with computer programming concepts will be helpful, but is not required.

Keywords

Keywords are predefined words used to communicate specific meanings to the compiler. For example, in C, a commonly used keyword isint, which is used to represent an integer. You can declare a variable of type integer in the following way:

int rabbits;

Here, int is a keyword that indicates rabbits is a variable of type int (integer). Probably we would use it to count, understand, or reproduce a number of rabbits.

There are a great many keywords you will use in C as you learn more, most notably char, float, double, while, andif.  All these keywords, their syntax, and applications will be covered in later sections.

Identifiers

In C, identifiers are used to give names to specific variables, functions, etc. Let's use our example from before:

int rabbits;

Remember that int indicated a variable of type integer. In that example we were counting rabbits, so rabbitsis the identifier. Let's check out another example:

int rabbits; float float;

The second declaration looks weird, right? That's because the programmer used an identifier that was the same as the keyword. This is not acceptable practice in C. We could fix this code by re-naming the identifier:

int rabbits; float carrots;

Looks better now! Here's a full list of rules for naming identifiers in C:

  • A valid identifier can have letters (upper or lowercase), digits, and underscores, e.g. Big_Rabbits3.
  • The first letter of an identifier should be either a letter or an underscore (so 3Big_Rabbits would not be acceptable).
  • You can't use keywords as an identifier.
  • There are no rules as to how long an identifier can be, but it's best to keep them descriptive and meaningful, but brief. For example, int Rabbits_That_I_Really_Like; is probably a bit excessive.
Variables

In C, variables are like a container used to store data. Imagine you're moving somewhere and you had a bunch of containers. It would probably be helpful to label each of them, right? It's the same with variables in C. In fact, you've already learned how to use an identifier to name a variable:

int rabbits;

In the above example, we declared an integer variable called rabbits. The name rabbits is actually just a symbolic representation of where the the container (memory) for rabbits is located on your computer. Declaring a variable has all sort of useful applications. Let's take a look at a simple one:

int rabbits = 50;

We've just declared that we have 50 rabbits stored at the location in our memory we've labeled rabbits. Cool! Let's continue:

int rabbits = 50; int carrots = rabbits - 1;

As you can see, we've now defined an additional integer variable called carrots that will always be 1 less than the number of rabbits. If the number of rabbits changes (and it can), so will the number of carrots. This is a very simple example, but maybe you're beginning to see how it could have important applications as you learn to build more and more complex programs.

Output

In C the function printf() is one of most important output functions. It sends formatted output to your screen. Let's see an example. If we run this program below:

#include < stdio.h > int main() { printf("I really like rabbits"); return 0; }

Then the program will output the following to our computer:

I really like rabbits

Again, this is a simple example, but it's one of the most important ways for us to get the computer to display information that it has returned. So how does this function work?

  • All programs in C must contain the main()function. Code execution begins from there.
  • To use printf() in our program, we include the < stdio.h > header file at the top of our code. Stdio is a kind of library containing useful C functions, so we have to list where we're getting the functions from.
  • The return 0; statement is a kind of exit for the program. It's technically optional in this program, but it's good practice to include it in your code.
